    ğŸš€ Getting Started¶

    Quick Start¶

    1. Navigate to docs directory: 

      cd docs

    2. Install dependencies: 

      # Using Python script
python setup_docs.py install

# Or using shell script
./start_docs.sh install

# Or manually
pip install -r requirements.txt

    3. Start the documentation server: 

      # Using Python script
python setup_docs.py serve

# Or using shell script
./start_docs.sh serve

# Or manually
mkdocs serve

    4. Open your browser and navigate to http://127.0.0.1:8000

    Alternative Methods¶

    Using Python Script¶

    cd docs
python setup_docs.py install  # Install dependencies
python setup_docs.py serve    # Start server
python setup_docs.py build    # Build for production
python setup_docs.py check    # Check setup

    Using Shell Script¶

    cd docs
./start_docs.sh install  # Install dependencies
./start_docs.sh serve    # Start server
./start_docs.sh build    # Build for production
./start_docs.sh check    # Check setup

    Manual Commands¶

    cd docs
pip install -r requirements.txt  # Install dependencies
mkdocs serve                     # Start server
mkdocs build                     # Build for production
